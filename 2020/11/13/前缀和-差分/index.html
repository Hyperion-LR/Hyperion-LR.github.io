前缀和

前缀和是一个数组的某项下标之前(包括此项元素)的所有数组元素的和。

简单来说：

a[0] = b[0];

a[1] = b[0] + b[1];

a[2] = b[0] + b[1] + b[2];

…

a[n] = b[0] + b[1] + b[2] + … + b[n];

对于a数组中的每一项a[i]都有a[i]等于b[i]前面项的和我们得到数组a为数组b的前缀和数组。
