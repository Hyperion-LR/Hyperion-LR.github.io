C++ 类&对象

作为面向对象编程语言的核心特性，类用于指定对象。类中包含数据的表示方式与行为方式。类中的数据与方法被称为类的成员。

C++ 类的定义

在C++中类的定义如下：

在C++中一个类的基本结构如下。其中，访问范围分为：

public:公有成员（类中的外部接口允许类的外部访问）

private:私有成员 （只允许本类中的函数访问，而类外部的任何函数都不能访问）

protected:保护成员（与private类似，差别表现在继承与派生时）。

成员变量的定义与普通变量没什么不同，只因该变量在类的内部，属于该类的成员故称其为成员变量。我们一般将其设为“private”防止其在外部访问的过程中发生改变。

一个类的成员函数之间可以互相调用。类的成员函数可以重载，也可以设定参数的默认值

成员变量就代表对象的“属性”，成员函数就代表对象的“方法”。成员变量和成员函数出现的先后次序没有规定。

成员函数在类的内部声明之后可以在类的外部进行定义。在定义的过程中成员函数民前要加上类名并用”::”作用域解析运算符来指明该成员函数所属的类。如下：
